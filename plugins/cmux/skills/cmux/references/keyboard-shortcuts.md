# cmux Keyboard Shortcuts

All keyboard shortcuts available in cmux, organized by category.

## Workspaces

Workspaces live in the sidebar. Each workspace has its own set of panes and surfaces.

| Action | Shortcut |
|---|---|
| New workspace | `Cmd+N` |
| Jump to workspace 1-8 | `Cmd+1` - `Cmd+8` |
| Jump to last workspace | `Cmd+9` |
| Close workspace | `Cmd+Shift+W` |
| Rename workspace | `Cmd+Shift+R` |

## Surfaces

Surfaces are tabs inside a pane.

| Action | Shortcut |
|---|---|
| New surface | `Cmd+T` |
| Previous surface | `Cmd+Shift+[` |
| Previous surface (alt) | `Ctrl+Shift+Tab` |
| Jump to surface 1-8 | `Ctrl+1` - `Ctrl+8` |
| Jump to last surface | `Ctrl+9` |
| Close surface | `Cmd+W` |

## Split Panes

| Action | Shortcut |
|---|---|
| Split right | `Cmd+D` |
| Split down | `Cmd+Shift+D` |
| Focus pane directionally | `Option+Cmd+Arrow` |
| Split browser right | `Option+Cmd+D` |
| Split browser down | `Option+Cmd+Shift+D` |

## Browser

| Action | Shortcut |
|---|---|
| Open browser surface | `Cmd+Shift+L` |
| Focus address bar | `Cmd+L` |
| Forward | `Cmd+]` |
| Reload page | `Cmd+R` |
| Open Developer Tools | `Option+Cmd+I` |

## Notifications

| Action | Shortcut |
|---|---|
| Show notifications panel | `Cmd+Shift+I` |
| Jump to latest unread | `Cmd+Shift+U` |
| Trigger flash | `Cmd+Shift+L` |

## Find

| Action | Shortcut |
|---|---|
| Find | `Cmd+F` |
| Find next | `Cmd+G` |
| Find previous | `Cmd+Shift+G` |
| Hide find bar | `Cmd+Shift+F` |
| Use selection for find | `Cmd+E` |

## Terminal

| Action | Shortcut |
|---|---|
| Clear scrollback | `Cmd+K` |
| Copy (with selection) | `Cmd+C` |
| Paste | `Cmd+V` |
| Increase font size | `Cmd++` |
| Decrease font size | `Cmd+-` |
| Reset font size | `Cmd+0` |

## Window

| Action | Shortcut |
|---|---|
| New window | `Cmd+Shift+N` |
| Settings | `Cmd+,` |
| Reload configuration | `Cmd+Shift+R` |
| Quit | `Cmd+Q` |
